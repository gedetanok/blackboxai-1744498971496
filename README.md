# Tech Blog

## Project Overview
Tech Blog is a modern web application designed to enhance technology literacy for Indonesian youths. It provides easy-to-understand information about the latest technology news, updates, guidelines, and reviews. The website features a responsive design, ensuring accessibility across various devices while offering a pleasant user experience.

## Installation
To set up the Tech Blog project locally on your machine, follow these instructions:

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   ```
2. **Navigate to the project directory**
   ```bash
   cd tech-blog
   ```
3. **Open `index.html` in your preferred web browser.**

No additional installation is required since it uses CDN links for styles and scripts.

## Usage
Once you open `index.html`, you will be greeted with a homepage highlighting various sections:

- **Home**: Overview and introduction to Tech Blog.
- **Updates, News, Recap, and About Page**: Information architecture for navigation.
- **Various Content Sections**: Including Technology Updates, Guides, Weekly Recaps, Articles, and more.

Simply navigate through the links and buttons to explore the features.

## Features
- **Responsive Design**: Works seamlessly on mobile and desktop.
- **Smooth Scrolling**: Enhanced user navigation through smooth scrolling behaviors.
- **Animated Buttons and Links**: Modern visual effects on hover to improve user engagement.
- **Content Variety**: Tech updates, guides, and weekly recaps make it diverse and informative.

## Dependencies
This project utilizes external libraries and frameworks:
- **Tailwind CSS**: For utility-first CSS components and responsive styling.
- **Font Awesome**: For providing scalable vector icons that can be customized.
- **Google Fonts**: For custom web typography, specifically the Inter font.

## Project Structure
The project consists of the following structure:
```
tech-blog/
│
├── index.html         # Main HTML file containing the structure and content of the website.
```
- **HTML File**: Contains all the markup for the website, inline CSS, and scripts.
